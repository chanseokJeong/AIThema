# AI 테마 뷰 (AI Thema View) - 컨셉 및 요구사항

## 1. 프로젝트 목표
주식 시장 데이터를 기반으로 "테마 강도 지수"를 시각화하는 웹 애플리케이션 (로컬 스탠드얼론).

## 2. 시스템 아키텍처 (로컬 단독 실행)
### A. 로컬 데이터 엔진 (Local Data Engine)
*   **뉴스 수집기 (Advanced)**:
    *   **소스**: 네이버 금융, 다음 금융 등 주요 포털의 증권 뉴스 통합 수집.
    *   **전처리**: 중복 기사 제거.
*   **AI 분석기 (LLM)**:
    *   **역할**: 테마 추출 + **필터링 (찌라시/가짜뉴스/중복 제거)**.
    *   **Prompt**: "다음 뉴스들 중 신뢰할 수 있는 팩트 기반의 뉴스만 선별하고, 핵심 테마와 관련 종목을 추출하라."

## 3. 핵심 로직
*   **테마 점수**: `Sum(종목 등락률) / Count(종목 수)`.
*   **AI 필터링**: 단순 키워드 매칭이 아닌, LLM의 문맥 이해 능력을 활용하여 광고성/자극적 기사 배제.

## 4. 기술 스택
*   **Backend**: Node.js + Express.
*   **AI**: Google Gemini API (권장) 또는 OpenAI API.
*   **Crawling**: Cheerio + Axios (User-Agent Rotation).
